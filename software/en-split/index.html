说明：音标采用“内置常见词词典 + 离线规则回退”生成。常见词更准确，生僻词和专有名词可能只提供近似读音。

分词说明：“提取短语”默认关闭；只提取单词时启动更快。短语提取基于离线短语词典命中，只有词典里已有的 2-3 词短语才会被识别。

词根记忆说明：当前仅对单词展示“词根记忆”列，优先使用内置精确词条；未命中时再尝试轻量前后缀拆分提示。短语这一列默认留空。

表格复制说明：结果区按表格展示；点击“复制结果”会按当前显示列顺序、列显隐和当前排序状态，以 Excel 友好的制表符分列格式复制；点击“下载 TSV”可直接用 Excel 打开。

未找到说明：`待补充释义` 的项目会优先排在前面，并用浅红色整行标出，便于优先处理。

在线补充说明：点击“翻译”后，仅会把离线未命中的单词或短语发送到所选在线服务，并在每次在线请求前做随机节流等待。`MyMemory` 可直接试用；`Google Cloud Translation API` 需要官方 API Key；`LibreTranslate` 公共站点通常也需要 API Key，内网部署时建议改成你自己的兼容服务地址。如果在线返回值与原英文词或短语相同，会视为未找到，不会覆盖 `待补充释义`。

缓存说明：在线查询结果会自动保存到 `data/catch.json`，刷新页面后会自动载入并优先复用。

已忽略单词说明：项目根目录下的 `ignore.txt` 支持手动编辑，刷新页面后会自动载入到“已忽略单词”输入框。

忽略文件说明：`ignore.txt` 里的单词会作为“已忽略单词”来源参与分词过滤，也可在页面里临时补充或修改。

词典已拆分为独立 JSON 文件。为确保浏览器能正常加载本地词典，建议通过静态文件服务访问，而不是直接以 `file://` 方式打开。
